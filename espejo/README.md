# RAULI-VISION — Servidor espejo (El Espejo)

Servidor VPS exterior: búsqueda optimizada, video curado (yt-dlp + ffmpeg), IA (resumir internet). Todas las respuestas comprimidas con Brotli.

**Stack previsto:** Go (o Node + Fastify), Redis (cola + caché), yt-dlp + ffmpeg, LLM API.
