
# Guía de Operador – Cuba Stealth

Objetivo:
- Mantener operación estable sin intervención técnica.

Reglas:
- No forzar descargas fuera de ventana.
- Usar prioridades solo cuando sea necesario.
- Revisar panel de diagnóstico local ante fallos.

Escenarios:
1) Sin señal → esperar (reanudación automática).
2) Señal débil → bajar prioridad.
3) Datos caros → usar ventana nocturna.

Soporte:
- Exportar snapshot local (sin red).
