Política de Privacidad - RAULI-VISION

Última actualización: Febrero 2026

1. Información que recopilamos

RAULI-VISION no recopila datos personales identificables. La aplicación funciona principalmente de forma local y utiliza conexiones a servicios externos (búsqueda, proxy) sin almacenar información personal del usuario.

2. Uso de datos

No recopilamos, almacenamos ni compartimos datos personales con terceros. El caché local se utiliza solo para mejorar el rendimiento y reducir el consumo de datos.

3. Permisos

La aplicación solicita permiso de Internet para realizar búsquedas y conectar con los servicios necesarios. No accedemos a contactos, ubicación ni otros datos sensibles.
